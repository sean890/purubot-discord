**`~g info`**
Search for genshin related information with an ID or keyword!

***Parameters***
`[keyword]`: Search for keywords of the info you wish to display
_OR_
`[ID]`: ID for the keyword of the info you wish to display

***Usages***
`~g info [keyword]`
_OR_
`~g info [ID]`

***Examples***
`~g info artifacts possible base stats`/`~genshin info 1`
`~g info artifact fodder exp`/`~genshin info 2`

***Some Keywords available***
For the complete list, type `~g infolist`.
